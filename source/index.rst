Welcome to VitalDSP's Documentation!
====================================

Welcome to the official documentation for the VitalDSP library! This library provides comprehensive tools for digital signal processing, focusing on filtering and transformations, especially in healthcare and biomedical applications.

Getting Started
===============

Whether you are a beginner or an experienced user, this documentation will guide you through the essential features of VitalDSP, offering examples and explanations for each module.

=======

Below are the core modules of the VitalDSP library:
